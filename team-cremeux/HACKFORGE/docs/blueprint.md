# **App Name**: FinanceFlow AI

## Core Features:

- Secure Authentication: Allow freelancers to create accounts, log in, and manage their profiles securely.
- Income and Expense Tracking: Enable users to log their income and expenses with categorization.
- Automated Tax Helper Tool: Provide AI-driven insights into potential tax savings. The AI will decide, using available tools, which saving suggestions might apply to each user, along with a projected tax summary, displayed using intuitive progress bars and charts.
- Scam Alerts: Deliver alerts for scam contracts using crowdsourced and AI-based rules, to warn freelancers in advance.
- Finance Chatbot: Provide access to a chatbot powered by AI, which will answer natural-language questions related to budgeting, tax, and scam prevention.

## Style Guidelines:

- Primary color: Deep sky blue (#3299CC) to represent trust and financial clarity.
- Background color: Very light blue (#E5F4FB) for a calm, professional feel.
- Accent color: Forest green (#228B22) to highlight positive financial actions and growth.
- Font: 'Inter', a grotesque-style sans-serif, should be used in headlines and body text due to its modern, objective, and neutral appearance, making it suitable for a financial application.
- Use a modern, card-based layout with soft gradients to provide a user-friendly interface on all devices. Focus on clean, minimal design for clarity.
- Implement subtle transitions and animations when loading financial data to provide smooth UX. A simple loading animation will provide feedback during AI tasks.
